# Termux-Bot

It's a simple discord bot that can execute commands to your termux.

## Try it for yourselves

#### Requirements

```
apt install git
apt install npm
apt install openssh
```

#### Installation

```
sshd
ssh localhost -p 802
git clone https://github.com/gx0c/termux-bot.git
cd termux-bot
npm install

#Copy your bot token here (TOKEN=your-token)
nano/vim/emacs .env
```

#### Running

```
node index.js
```
